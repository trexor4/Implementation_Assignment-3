# Implementation_Assignment-3
---

## Requirements

- Python 3.8+ (or equivalent)
- Python packages:
  - `pulp`
  - `numpy`
  - `pandas`
  - `matplotlib`

Install dependencies with:

```bash
python -m pip install pulp numpy pandas matplotlib
```

---

## Running the code

From this directory, run:

```bash
python problem1.py
python problem2.py
```

- `problem1.py` prints the optimal line parameters and saves `regression_plot.png`.
- `problem2.py` reads `Corvallis.csv`, prints the fit parameters, and saves `temperature_fit.png`.
